根据这个Jekyll模板的结构，您需要修改以下几个关键部分来个性化网站内容：

## 1. 修改基本站点配置

编辑 _config.yml 文件，更改以下内容：

# 站点基本信息
title: 您的网站标题
description: '您的网站描述'
url: "https://您的用户名.github.io"  # 如果使用GitHub Pages

# 作者信息
author: 您的姓名
author-pic: 您的照片文件名  # 放在assets/img/目录下
about-author: 您的个人简介

# 联系方式
email: 您的邮箱
website: 您的个人网站
linkedin: 您的LinkedIn用户名
github: 您的GitHub用户名
twitter: '@您的Twitter用户名'

## 2. 替换品牌图片

将以下文件替换为您自己的图片：
- /assets/img/branding/MVM-logo-full.svg - 主要logo
- /assets/img/branding/MVM-logo-full-dark.svg - 深色模式logo
- /assets/img/branding/MVM-symbol.svg - 简化logo
- /assets/img/favicon/favicon.ico - 网站图标

## 3. 添加您的博客文章

在 _posts 目录中：
- 删除现有的示例文章
- 创建新文章，命名格式：YYYY-MM-DD-文章标题.md
- 每篇文章的开头添加前置元数据：

---
layout: post
title: 文章标题
date: 2024-01-01 12:00:00
description: 文章描述
img: 图片文件名  # 放在assets/img/posts/目录下
tags: [标签1, 标签2]
author: 您的姓名
toc: yes  # 是否显示目录
---

这里是文章内容，使用Markdown格式编写...

## 4. 更新静态页面

修改 _pages 目录中的页面：
- about.html - 关于页面
- archive.html - 归档页面
- tag.html - 标签页面
- tags.html - 标签列表页面

## 5. 自定义样式和颜色

编辑 assets/css/main.css 文件中的CSS变量：

html {
  --accent: #您的主色调;
  --main: #您的主要颜色;
  --text: #文本颜色;
  --background: #背景色;
}

## 6. 配置评论和统计

如果您想使用评论和统计功能，在 _config.yml 中配置：
- comments: 评论系统设置
- analytics: Google Analytics ID
- mailchimp: 邮件订阅设置

## 7. 测试和部署

本地测试：
npm run dev

确认一切正常后，推送到GitHub，GitHub Pages会自动部署。
